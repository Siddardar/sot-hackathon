## Installation

Follow these steps to run the project on your machine.

### 1. Prerequisites
Make sure you have **Node.js** (version 18 or later) installed.  
Download it here: [nodejs.org](https://nodejs.org)

Check your version by running:
```bash
node --version
npm --version
```

### 2. Install dependencies

```bash
cd path/to/your-project-folder
npx create-next-app@latest leak-app --typescript --tailwind --eslint --app --no-src-dir --import-alias "@/*"
npm install
npm run dev
```
